# 📊 Exam Score Prediction Dashboard

## 🚀 Project Overview

This project presents an interactive **Power BI dashboard** designed to analyze student performance and uncover key factors influencing exam outcomes. The analysis focuses on how **study behavior, resource accessibility, and exam structure** impact student success, with the goal of enabling **data-driven academic decision-making**.

## Business Problem

Educational institutions often struggle to identify:

* Which study methods are most effective
* How access to resources (e.g., internet) affects performance
* What level of exam difficulty maximizes student outcomes
This project answers the question:

 **What are the key drivers of student performance, and how can they be optimized?**
##  Dashboard Preview

##  Analytical Approach

### 1. Data Collection

* Source: Kaggle (Student Performance Dataset)
* Structured dataset containing student demographics, behavior, and academic variables

### 2. Data Preparation

* Cleaned and standardized categorical variables
* Removed inconsistencies and duplicates
* Aggregated key metrics (students, study hours)

### 3. Data Visualization

Built an interactive Power BI dashboard including:

* KPI Cards (Total Students, Study Hours, Age)
* Treemap (Course Distribution)
* Bar Chart (Study Methods)
* Donut Chart (Gender Distribution)
* Performance Chart (Exam Difficulty vs Score)
* Area Chart (Internet Accessibility)

##  Key Insights
**Self-study and online learning are the most adopted methods**, indicating a shift toward independent learning
**Internet access plays a critical role**, with a significant gap between students with and without access
 **Moderate exam difficulty yields the highest performance**, suggesting optimal challenge improves outcomes
 **Gender distribution is balanced**, showing no strong performance bias

## Recommendations

* Expand access to **digital learning resources**
* Encourage **self-directed learning frameworks**
* Design assessments with **moderate difficulty levels**
* Provide **targeted support for complex subjects**
